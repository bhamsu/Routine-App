
INTRODUCTION
  It is a cross-platform desktop App named Routine Management App, built with
  Node.js and Electron, with python as a backend and Google's Firestore as the
  Cloud database, and .json file for local database.

REQUIREMENTS
  1. Node.js and npm [https://nodejs.org/en/]
  2. Python 3.6.x [https://www.python.org/]
